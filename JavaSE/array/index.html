数组

概念: 同一种类型数据的集合，其实数组就是一个容器
好吃：可以自动给数组中的元素从0开始编号，方便操作这些元素。

格式一： 元素类型［］ 数组名 ＝ new 元素类型[元素个数或者数组长度]
Eg: int[] arr = new int[5]

格式二：元素类型[] 数组名 ＝ new 元素类型[] {元素，元素，。。}
Eg: int[] arr = new int[] {1,3,4,5,};
int [] arr = {1,3,5,7}; //简写

内存结构(重要)

java在运行时，需要在内存中分配空间。为了提高运行效率，有对空间进行了不同区域的划分，因此每一片区域都有特定的处理数据的方式和内存管理方式。

栈内存 : 用于存储局部变量，当数据用完，所占空间自动释放。

堆内存: - 数组和对象，通过new建立的实列都放在堆内存中。 - 每一个实体都有内存地址. - 实体中的变量都有默认初始化值。 - 实体不在被使用，会在不确定的时间被垃圾回收器回收。
